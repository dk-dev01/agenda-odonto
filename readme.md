# 🦷 Agenda Odontológica API

Uma API REST desenvolvida com **Java, Spring Boot e React** para gerenciar uma clínica odontológica. O sistema tem como objetivo facilitar o cadastro e o gerenciamento de pacientes, dentistas, consultas e demais informações relacionadas ao atendimento odontológico.

Este projeto está sendo desenvolvido seguindo boas práticas de desenvolvimento back-end, arquitetura em camadas e princípios de código limpo, servindo tanto como projeto de aprendizado quanto como portfólio profissional.

---

# 📌 Objetivos

- Desenvolver uma API REST completa utilizando Spring Boot.
- Aplicar boas práticas de arquitetura em projetos Java.
- Aprender e utilizar o ecossistema Spring.
- Trabalhar com persistência de dados utilizando JPA/Hibernate.
- Realizar validações de dados.
- Implementar tratamento global de exceções.
- Consumir a API utilizando uma aplicação Front-end em React.
- Versionar o projeto utilizando Git e GitHub.
- Preparar a aplicação para deploy em ambiente de produção.
